About the Driver Drowsiness Detection System

The Driver Drowsiness Detection System is an intelligent safety solution designed to monitor drivers in real-time and detect signs of drowsiness using computer vision and machine learning techniques.

Purpose

Road accidents caused by driver fatigue are a major concern worldwide. This system aims to prevent such incidents by alerting the driver when signs of drowsiness—such as prolonged eye closure or yawning—are detected.

Key Features

  • Real-time monitoring using webcam input
  • Eye-blink and yawn detection using facial landmarks
  • Sleep count and session duration tracking
  • Session dashboard with visual data representation
  • Location tracking (optional) at session start
  • User login and registration system

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (Flask)
  • Database: MongoDB
  • Computer Vision: OpenCV, Dlib

Use Case

This system is ideal for individual drivers, transportation companies, or research institutions seeking to enhance road safety through AI-driven monitoring systems.

Note: This project is developed as part of an academic initiative to demonstrate the integration of web technologies with real-time computer vision applications.
